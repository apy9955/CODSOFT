# Spam SMS Detection

## 📌 Objective
Build a Machine Learning model to classify SMS messages as Spam or Ham using Natural Language Processing (NLP).

## 📂 Dataset
- SMS Spam Collection Dataset
- Total Records: 5572

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 🤖 Machine Learning Algorithm
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## 📊 Results
- Accuracy: 96.62%
- Precision (Spam): 1.00
- Recall (Spam): 0.76
- F1-Score: 0.86

## 📁 Project Files
- Spam_SMS_Detection.ipynb
- spam_sms_model.pkl
- tfidf_vectorizer.pkl
- requirements.txt

## 🚀 How to Run

1. Install the required libraries.
2. Open the notebook.
3. Run all cells.
4. Test the model with custom SMS messages.
