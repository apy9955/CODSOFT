# Movie Genre Classification

## Objective
Build a Machine Learning model to predict the genre of a movie based on its plot description.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Algorithm
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Dataset
IMDb Movie Genre Classification Dataset

## Results
- Official Test Accuracy: 51.88%
