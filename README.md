# CODSOFT Machine Learning Internship

This repository contains the Machine Learning tasks completed as part of the CodSoft Internship.

## 📂 Completed Tasks

### 🎬 Task 1: Movie Genre Classification
- Predicted movie genres using NLP techniques.
- Algorithm: TF-IDF + Multinomial Naive Bayes

### 💳 Task 2: Credit Card Fraud Detection
- Detected fraudulent credit card transactions.
- Algorithms: Logistic Regression and Random Forest
- Best Model: Random Forest

### 📩 Task 4: Spam SMS Detection
- Classified SMS messages as Spam or Ham.
- Algorithm: TF-IDF + Multinomial Naive Bayes

## 🛠️ Technologies Used

- Scikit-learn
- Matplotlib
- Seaborn
- Joblib
